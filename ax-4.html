04

Git과 GitHub의 차이

Git
🗂️
버전 관리 도구
코드의 변경 이력을 추적하는 도구예요.

"저장 버튼 + 히스토리 기능"이라고 생각하면 됩니다.
GitHub
☁️
클라우드 저장소
Git으로 관리하는 코드를 온라인에 올려두는 저장소예요.

협업 기능(코드 리뷰, 이슈 관리, 프로젝트 보드 등)이 더해진 플랫폼입니다.

06

Step 0 — GitHub Repository 만들기

1
github.com 접속 후 로그인
https://github.com
2
New repository 클릭
우측 상단 + 버튼 또는 프로필 페이지에서 생성
3
Repository 이름 입력 후 Create
Public으로 설정해야 GitHub Pages로 공개 호스팅 가능
4
Settings → Pages → Branch 설정
main 브랜치, / (root) 선택 후 Save → 호스팅 URL 발급
07

단일 파일 올리기 3가지 방법

📝
방법 1
MD 파일 복붙
Repository에서 Create new file 클릭 → 파일명 입력 → 내용 붙여넣기 → Commit
💻
방법 2
Gemini HTML 복붙
Gemini에서 생성한 HTML 코드를 복사 → GitHub 새 파일로 붙여넣기 → index.html로 저장
🚀
방법 3
Antigravity HTML 업로드
Antigravity에서 만든 HTML 파일 → Repository에서 Upload files 클릭 → 드래그 업로드
💡
파일명을 index.html로 저장하면 GitHub Pages URL에 파일명 없이 바로 접속됩니다.

Step 4 — AI에게 Push 시키기

언제 사용?
  • 여러 파일이 폴더로 구성되어 있을 때
  • Antigravity 또는 VS Code에서 작업 중일 때
  • 터미널 명령어가 익숙하지 않을 때
어떻게?
🤖
AI에게 Repository URL을 알려주고 Push 요청!
AI에게 이렇게 말하세요
"https://github.com/내아이디/저장소이름" 여기에 push 하고 싶어 해줘~~~~~
💡
AI가 SSH 연결 확인부터 git add, commit, push 명령어까지 전부 대신 실행해 줍니다.
11

Step 5 — 직접 Push 하기 (터미널)

1
SSH 키 생성
엔터 × 2, 비밀번호 입력
2
GitHub에 공개키 등록
프로필 → Settings → SSH and GPG keys → New SSH key → 붙여넣기
3
연결 확인
ssh -T git@github.com
4
AI에게 Repository 알려주고 Push
VS Code / Antigravity 터미널에서
SSH 키 생성 명령어
# 1. SSH 키 생성 ssh-keygen -t ed25519 -C "your_email@example.com" # 엔터 엔터, 비밀번호 입력하기 # 2. 공개키 확인 (복사해서 GitHub에 붙여넣기) cat ~/.ssh/id_ed25519.pub # 3. 연결 확인 ssh -T git@github.com # 4. Push 요청 예시 "https://github.com/내아이디/저장소 여기에 push 해줘"
